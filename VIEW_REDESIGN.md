# 🎨 View the New Redesigned Studio Y7 Website

## Quick Start

### 1. Make sure backend is running
```bash
cd "S:\mern project\Studio-Y7\backend"
npm run dev
```

### 2. Start the frontend (in new terminal)
```bash
cd "S:\mern project\Studio-Y7\client"
npm run dev
```

### 3. Open in browser
```
http://localhost:5173
```
